### Usage

To enable the screenshot tool, add to your config:

```json
{
  "tools": {
    "screenshot": {
      "enabled": true
    }
  }
}
```

Then use in chat:
- `screenshot(mode="full")` - Capture full screen
- `screenshot(mode="window")` - Select a window
- `screenshot(mode="area")` - Drag to select area
- `screenshot(mode="full", send=true)` - Capture and send to current session
